# Wenshu 记忆系统说明

## ⭐ 统一记忆层（推荐使用）⭐

从现在开始，推荐使用 `memory.py` 作为统一入口！

```python
from memory import (
    # 短记忆
    short_term_memory,
    # 长记忆（所有功能）
    append_teaching_feedback,
    update_learning_progress,
    get_memory_summary,
    search_memories,
    get_memory_stats,
    add_memory_tags,
    consolidate_memories,
    cleanup_old_memories,
    # 统一上下文
    get_full_context,
)
```

### 快速使用统一记忆层

```python
from memory import short_term_memory, get_full_context

# ── 短记忆（会话级）────────────────────────────────────

# 对话历史
short_term_memory.append_message("student_123", "student", "你好")
short_term_memory.append_message("student_123", "bot", "来做题吧")
messages = short_term_memory.get_recent_messages("student_123", limit=10)

# 推题状态
short_term_memory.set_last_pushed_ids("student_123", [101, 102, 103])
ids = short_term_memory.get_last_pushed_ids("student_123")

# 自定义键值
short_term_memory.set("student_123", "current_mode", "quiz")
mode = short_term_memory.get("student_123", "current_mode")

# ── 获取完整上下文（短+长）───────────────────────────

ctx = get_full_context("student_123")
# {
#   "short_term": { ... },           # 短记忆状态
#   "recent_messages": [ ... ],      # 最近对话
#   "long_term_summary": "..."       # 长记忆摘要
# }

# ── 长记忆（持久化）────────────────────────────────────

# 追加教学反馈
append_teaching_feedback(
    student_id="student_123",
    insight="学生对假设检验理解较弱",
    student_name="张三"
)

# 搜索记忆
results = search_memories("student_123", "关键词")

# 获取统计
stats = get_memory_stats("student_123")
```

## 模块说明

### `memory.py` - 统一入口（推荐）✅

整合了短记忆和长记忆的所有功能，提供统一的 API。

### `memory_store.py` - 长记忆底层

仅在需要直接访问底层功能时使用，一般使用 `memory.py` 即可。

## 快速开始（详细版）

### 短记忆（会话级）

```python
from memory import short_term_memory

# 存取键值
short_term_memory.set("student_123", "temp_flag", "value")
val = short_term_memory.get("student_123", "temp_flag")

# 对话历史
short_term_memory.append_message("student_123", "student", "你好")
short_term_memory.append_message("student_123", "bot", "来做题吧")
messages = short_term_memory.get_recent_messages("student_123", limit=10)

# 推题状态
short_term_memory.set_last_pushed_ids("student_123", [101, 102, 103])
ids = short_term_memory.get_last_pushed_ids("student_123")

# 清空
short_term_memory.clear("student_123")
```

### 长记忆（持久化）- 基础功能

```python
from memory import (
    append_teaching_feedback,
    update_learning_progress,
    get_memory_summary,
)

# 追加教学反馈
append_teaching_feedback(
    student_id="student_123",
    insight="学生对假设检验理解较弱，需要加强练习",
    student_name="张三"
)

# 更新学习进展
update_learning_progress(
    student_id="student_123",
    update_text="已完成第五章练习题，正确率 85%",
    student_name="张三"
)

# 获取长记忆摘要（注入 LLM 上下文）
summary = get_memory_summary("student_123")
```

### 长记忆（持久化）- 增强功能 ✨

```python
from memory import (
    search_memories,
    score_memory_importance,
    get_memory_stats,
    add_memory_tags,
    get_memories_by_tag,
    consolidate_memories,
    cleanup_old_memories,
)

# 1. 搜索记忆
results = search_memories("student_123", "关键词")
for r in results:
    print(f"{r['type']}: {len(r['matches'])} 个匹配")

# 2. 获取记忆统计
stats = get_memory_stats("student_123")
print(f"总记忆数: {stats['total_memories']}")
print(f"最后更新: {stats['last_updated']}")

# 3. 评分记忆重要性
score = score_memory_importance("student_123", "teaching_feedback")
print(f"重要性评分: {score:.2f}")

# 4. 添加标签
add_memory_tags("student_123", "teaching_feedback", ["重要", "薄弱点", "AP统计"])

# 5. 按标签查询
tagged_memories = get_memories_by_tag("student_123", "重要")

# 6. 整合记忆（使用 LLM 去重、压缩）
consolidated = consolidate_memories("student_123")

# 7. 清理过期记忆
result = cleanup_old_memories("student_123", days=90)
print(f"归档: {result['archived']}, 删除: {result['deleted']}, 跳过: {result['skipped']}")
```

## API 参考（完整列表）

### 统一入口 - `memory.py`

| 函数/对象 | 说明 |
|----------|------|
| `short_term_memory` | 短记忆单例对象 |

### 短记忆方法 - `short_term_memory.*`

| 方法 | 说明 |
|------|------|
| `set(student_id, key, value)` | 设置键值 |
| `get(student_id, key, default)` | 获取键值 |
| `delete(student_id, key)` | 删除键 |
| `clear(student_id)` | 清空所有短记忆 |
| `get_state(student_id)` | 获取完整状态 |
| `append_message(student_id, role, content, max_limit)` | 追加对话消息 |
| `get_recent_messages(student_id, limit)` | 获取最近对话 |
| `clear_messages(student_id)` | 清空对话历史 |
| `set_last_pushed_ids(student_id, question_ids)` | 设置推送ID列表 |
| `get_last_pushed_ids(student_id)` | 获取推送ID列表 |
| `set_current_session_id(student_id, session_id)` | 设置会话ID |
| `get_current_session_id(student_id)` | 获取会话ID |

## 重要性评分规则

`score_memory_importance()` 根据以下规则评分：

- **基础分**: 0.5
- **近期更新**: +0.2 (7 天内) / +0.1 (30 天内)
- **关键词**: +0.05/个（错误、错题、不会、困难、掌握、进步、重要、关键等）
- **标记**: +0.2 (`important=true`) / +0.1 (`consolidated=true`)
