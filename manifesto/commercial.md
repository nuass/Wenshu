# Commercial — 商业路径

## 市场判断

**K12 课外培训市场**在监管后加速向小班/一对一家教转移。
家教老师是个体经营者——他们有强烈的提效需求，但没有企业预算采购 SaaS。

这意味着：**定价必须低，价值必须即时可见。**

飞书在教育机构有相当渗透率（尤其是中等规模机构），
这是我们的天然分发渠道，不需要教用户装新工具。

---

## 客户画像

**核心用户：独立家教老师 / 小型教培工作室（2-10 人）**

- 同时辅导 10-50 名学生
- 已在用飞书或微信与学生沟通
- 有 PDF 题库或教材，但靠手动发题
- 没有技术团队，需要开箱即用

**次级用户：中型教培机构**

- 需要多老师、多学科统一管理
- 需要学生数据报表给家长看
- 愿意为数据安全付更高费用（私有化部署）

---

## 商业化路径

### 阶段一：开源 + 云托管（当前）

```
开源核心代码（MIT）
   +
托管版：老师注册即用，无需部署
   ↓
定价：免费（≤3名学生）/ ¥99/月（≤20名学生）/ ¥299/月（无限制）
```

开源建立信任和开发者社区，托管版收取服务费。

### 阶段二：数据报告产品

```
学生学习周报（自动生成，可发给家长）
知识点掌握热力图
班级整体薄弱点分析
   ↓
家长愿意为"看见孩子进步"付费
```

### 阶段三：机构版（私有化）

```
私有部署 + 专属支持
多校区统一管理
与机构现有系统（排课、收费）对接
   ↓
年框合同，¥2-10万/年
```

## 为什么现在

- LLM 成本在过去 18 个月下降了 10 倍，个性化 AI 教学首次在经济上可行
- 飞书 Cardkit / lark-cli 生态成熟，技术门槛大幅降低
- 国内教育监管后，大量家教老师需要效率工具但市场供给不足
